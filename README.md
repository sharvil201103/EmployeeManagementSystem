# Employee Management System

This is a simple **Employee Management System** built using:

- C#
- WinForms
- LINQ
- NuGet Packages

## Features (Planned)
- Add Employee
- Edit Employee
- Delete Employee
- Search Employee using LINQ
- Logging using Serilog
- Optional: Save/Load employee data using JSON

## Setup
- Clone the repo
- Open in Visual Studio
- Run the project
